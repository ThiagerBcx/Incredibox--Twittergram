# Incredibox - Twittergram

CONCEPTO DEL PROYECTO

Twittergram es un mod inspirado en la estética Sprunki que fusiona el estilo visual de las redes sociales con la energía de los videojuegos retro. Es una propuesta diseñada para crear ritmos rápidos, caóticos y divertidos, manteniendo la esencia de los proyectos experimentales nacidos en Scratch.

BASES MUSICALES (FL STUDIO)

El sonido de este mod tiene su origen en composiciones específicas de FL Studio Mobile, las cuales definen su identidad 8-bit y pegajosa. Los archivos base utilizados para la conversión son:

Edwan - Permich - Tropical.flm: Aporta la base rítmica y el movimiento.

8-bit Tickle Toes.flm: Define la textura de videojuego pixel.

Shop: El archivo clave para los sonidos de interfaz y loops de transición.

CARACTERÍSTICAS PRINCIPALES

Estética Internet

El nombre surge de la unión de Twitter e Instagram, buscando esa identidad visual moderna pero procesada a través de un filtro de bits. No es un mod serio; es un espacio para jugar y crear mezclas extrañas.

Sonidos Retro y Pegajosos

Gracias a su origen en archivos .flm, los sonidos son fáciles de combinar. El objetivo es que cualquier usuario pueda generar loops "raros pero cool" de forma inmediata desde su navegador.
